## 一、串聯洞察（三階段演化）

### 階段 1：兩 IP 串連
colombo 原話：
> 「GlowFrame 解釋了造物森林中、為什麼人類留下一堆造物之後就不見了。因為人類都拋棄肉體、數位永生了。」

### 階段 2：HH 加入、三 IP 同源
colombo：
> 「HH 是 GlowFrame 寵物系統延伸出的獨立 project、把機獸從『玩家寵物』進化成『RL 自主協作生物』。」

### 階段 3：**雙盲反轉**（最大反轉、改寫整個時間軸）
colombo 原話：
> 「機獸是人類用戰甲回來探索地球後發現的物種。在與這些機獸對戰到訓服的過程就是 HH 的故事。
>
> 那這些機獸又是怎麼來的呢？原本造物森林的小動物又到哪去了呢？
>
> **原來人類因為核災拋棄地球、數位永生之後、蛙蛙一族受輻射影響智力有所提升**、所以才有了會用廢棄物打造裝備與掠食者對抗。
>
> 後來幾百年蛙蛙一族持續進化、有了自己的文明、科技水平也有巨大的提升、為了彌補自身肉體弱小的短版、**所以才模仿巨型野獸、打造出各種仿生機獸**。
>
> **人類不知道、如果把機獸的腦袋打開、會發現裡面有小蛙蛙坐在駕駛艙。
> 蛙蛙也不知道、這些從太空來的強大機甲、其實是以前的人類又回來地球。**」

→ **這個反轉重寫整個宇宙觀**：
1. **核災** = 三 IP 共同起源
2. **HH 機獸是蛙蛙造的、不是人類造的**
3. **戰甲跟機獸對戰 = 兩個物種互相不認識、且都不知對方存在於殼裡**
4. **HH 的「Harmonic」最終的真意 = 跨物種理解的可能性**

## 四、雙盲反轉：HH 的核心敘事力量

### 4.1 兩個物種、兩層殼、兩種誤解

```
人類陣營：
  意識（衛星）
   ↓ 下載
  戰甲（金屬殼）
   ↓
  地表

蛙蛙陣營：
  小蛙駕駛員
   ↓ 進入
  機獸（金屬殼）
   ↓
  地表

兩邊在地表遭遇：
  人類看到機獸 → 「強大野生生物」
  蛙蛙看到戰甲 → 「太空來的神秘巨人」
  兩邊都不知道對方殼裡有意識
  兩邊都不知道對方曾是地球的主人 / 倖存者
```

### 4.2 為什麼這個雙盲設計如此強

| 文化參照 | 跟「雙盲反轉」相似的時刻 |
|---|---|
| Pacific Rim | 人 vs 怪獸、但 Jaeger 裡有兩個 pilot 神經連結——「殼裡有意識」motif |
| Wall-E | 機器人有意識、但人類也已退化、誰才是「人」 |
| 黑客任務 Matrix | 真實 vs 虛擬、意識 vs 軀殼 |
| 阿凡達 | 意識下載至外星軀體、跟原住民衝突 |
| 普羅米修斯 | 創造者跟造物互相不認識 |
| **造物森林 × GlowFrame × HH** | **全部融合 + 添加「核災起源」 + 「雙物種雙盲」**——前所未有 |

### 4.3 玩法上能怎麼利用這個反轉

**第一階段（人類視角）**：
- 玩 HH、玩家是人類戰甲使用者
- 對「神秘機獸」進行對戰 / 馴服
- 機獸像 Horizon Zero Dawn 的機械獸——危險、神秘、強大
- 玩家以為這是「征服地球神秘生物」的故事

**第二階段（揭曉時刻）**：
- 某次戰鬥、玩家打壞一隻機獸的腦袋
- **發現裡面有駕駛艙、有一隻死掉的小蛙**
- 整個故事翻轉
- 玩家意識到：**「我們在獵殺的不是野獸、是另一個文明」**

**第三階段（蛙蛙視角）**：
- 解鎖蛙蛙陣營玩法
- 玩家操作蛙蛙駕駛員 / 機獸工程師
- 看見「太空巨人」入侵自己的文明
- 玩家意識到：**「在他們眼中、我們也是『神秘他者』」**

**第四階段（harmonic 階段）**：
- 雙方終於發現真相
- 開始合作而非戰爭
- HH 的「Harmonic」終於有了完整意義
- → **這是整個 LL 宇宙的高潮時刻**

### 4.4 雙盲反轉的核災 callback

```
紀元 -X：核災
  ├─ 人類逃離（拋棄肉體 + 上傳衛星）
  └─ 蛙蛙留下（受輻射 + 智力突變）

  兩條物種演化路徑：
  人類 → 數位 → 衛星上等待 → 戰甲下凡探索
  蛙蛙 → 肉體 → 地表掙扎 → 文明 → 機獸

  紀元 500：在地表相遇
  兩邊都是 → 核災的倖存者後裔
  兩邊都是 → 在金屬殼裡的脆弱意識
  兩邊都是 → 互相把對方當「他者」
  ────────────────────────────────────
  ↓
  真正的 boss 從來不是對方
  是當初讓人類拋棄地球的「核災」
  是讓兩物種無法理解彼此的「時間 + 隔離」
  是把「同源」變成「他者」的「遺忘」
```

→ **這個 narrative 經得起 Marvel / Pixar / 宮崎駿級的展開**。

## 七、實際操作含意

### 短期（不動）
- 兩 IP 設計文件已收（current state）
- 不動工、論文 + EE pivot 先

### 中期（半年-1年後）
- 若任一 IP 真要動工、要記得**保留統一時間軸**
- 不要讓兩 IP 內部設計衝突（例如戰甲不能出現在造物森林同時代）
- 美術 pipeline 可共用（voxel + MagicaVoxel）

### 長期（2-3 年後）
- 兩 IP 同時 ship 是奢望、但**兩 IP 在敘事上互相 reference** 是必要
- 第一個 ship 的 IP 末尾可 tease 另一個 IP
- LL 宇宙觀對外溝通時、把「兩切面同悲劇」當核心 talking point

## 八、為什麼這個串聯如此重要

5 年前 colombo 分別做這兩個 IP 時、應該沒想到能串連。
今天的串連洞察是**回頭看才看出來的結構性對位**——這是創作者最珍貴的時刻。

這種「**兩個獨立作品意外組成同一世界**」的例子歷史上有：
- Stephen King 《Dark Tower》系列把多個獨立小說串起來
- Marvel Cinematic Universe（但他們是計畫好的、不是回頭發現的）
- 宮崎駿《天空之城》、《魔法公主》、《幽靈公主》共同的環保母題

→ colombo 的 SS IP 宇宙剛好屬於前者——**自然長成的 unified universe**、不是設計出來的。
→ 這種「自然性」本身就是 marketing 故事。

## 九、結語

> 一個 IP 賣的是「**故事**」、
> 兩個 IP 串成的世界賣的是「**信仰**」、
> 三個 IP 的雙盲反轉賣的是「**真相揭曉的瞬間**」。
>
> LL 不是要做三款遊戲、是要建一個讓玩家**相信**、然後**重新理解**的世界。
> 而這個世界、源於 colombo 5 年前兩個沒打算串連的設計、加上最近做的 HH——
> 今天回頭看才發現它們本來就在**同一個核災倖存者的故事**裡。
>
> **HH 是中心、其他兩個 IP 是前史**——
> 但只有先玩過前史、才能感受 HH 揭曉時刻的力量。
