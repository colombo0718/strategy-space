## 重做選項（未來動工時）

| 工具 | 適合度 |
|---|---|
| **MagicaVoxel** | 原作工具、可保留 5 年前風格 |
| Goxel | 跨平台、現代化 |
| Blender voxel addon | 整合度高、可直接接遊戲引擎 |
| web-based（Voxel.js / Pixxen）| 對齊 LL「Vibe Coding」風格、瀏覽器內製作 |

→ 推薦：**MagicaVoxel + 對齊原作美感**、不要為了「現代化」改風格。

## 視覺風格定錨

從原版截圖看出的美感共識：
- **粗體素**（low poly voxel）、不是高解析
- **鮮明色彩**（綠 / 紅 / 黃 / 白 / 紫對比強）
- **武器跟角色等比放大**（美工刀片跟樹蛙差不多大）
- **黑色背景 + 細網格** = MagicaVoxel 預設環境

→ 未來重做時、不要追求寫實、追**重現童年積木感**。
