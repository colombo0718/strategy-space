# 造物森林（Creation Forest）

> SS 旗下第一個遊戲 IP、2021 年 colombo 原創、2026-05-27 復活進 SS。
>
> **核心 logline**：森林食物鏈中下層的小動物、用人類遺棄的小垃圾打造裝備、抵抗各種巨大掠食者。
> 大敘事是**反諷環境污染**——主角們的力量來自敵人。

最後更新：2026-05-27

## 一、定位

- **遊戲類型**：類魂（Souls-like）+ 可愛動物
- **玩家視角**：食物鏈中下層生物
- **核心 mechanic**：拾荒組裝（人類垃圾 → 武器 / 防具）
- **敘事母題**：環境污染反諷（你的力量來自摧毀你棲地的人類）
- **平台目標**：未定（可 web / 桌面 / mobile、看實作時機）
