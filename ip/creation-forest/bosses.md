# 造物森林：Boss 進化結構

## 設計原則

- **每隻角色的天敵都成立**：對應真實食物鏈
- **三階梯遞進**：低 → 中 → 文明本身
- **最終 boss = 環境本身**：不是某隻怪、是「**人類社會的存在**」
- 整個 boss 設計呼應**反諷環境污染**主敘事

---

## 第一階：低階掠食者（新手村 → 中期）

「**訓練玩家了解自己物種的弱點**」階段。

| 對應主角 | Boss | 教學重點 |
|---|---|---|
| 小蛙 | 青蛇 | 近戰躲閃 + 蛇咬節奏 |
| 小蛙 | 鷸鳥 | 飛行 + 尖嘴垂直攻擊閃避 |
| 阿龍變色龍 | 老鷹 | 隱身有極限（從天而降）|
| 阿龍 | 狐狸 | 嗅覺強、隱身要逆風 |
| 阿龍 | 巨蟒 | 熱感應、隱身無效、逼學物理戰 |
| 松松松鼠 | 貓頭鷹 | 夜間靜音 + 被動偵測 |
| 松松 | 家貓 | 速度 + 玩弄獵物連續閃避 |
| 松松 | 黃鼠狼 | 鑽洞、無處可躲、戰略遷移 |
| 金絲雀 | 紅尾鵟 | 猛禽空中對戰 |
| 金絲雀 | 流浪貓 | 家養 PTSD + 速度 |
| 小螳螂 | 蝙蝠 | 超音波、擬態無效 |
| 小螳螂 | 蜂虎 | 專吃飛蟲、空中對戰 |
| 小螳螂 | 大型蜘蛛 | 陷阱對決、線 vs 線 |

→ 每個角色都有 2-3 隻專屬 boss、共 13 隻新手村級。
→ 玩家透過這階段**學會自己角色的優勢 + 弱點**。

---

## 第二階：中階掠食者（混合區）

「**單一物種應付不來、開始要組隊**」階段。

雜食 / 跨食物鏈 / 大型生物：

| Boss | 為什麼難 | 設計重點 |
|---|---|---|
| 狐狸（升級版）| 5 隻都吃、智力高、會挖洞 | 多角色配合追蹤 |
| 浣熊 | 雜食 + 翻垃圾 + 跟人類關係近 | 跟主題雙關、會搶玩家撿到的垃圾 |
| 巨蟒 | 全角色都怕、可吞食小型角色 | 多角色協同攻擊 |
| 蒼鷺 | 大型涉禽、水陸雙棲 | 把小蛙從水池逼出來 |
| 黃鼠狼群 | 群體狩獵、智力高 | 戰術分頭、不能單打 |
| 大型鳥猛禽 | 視覺極佳、攻擊範圍大 | 隱身對牠效率有限 |

→ 約 8-10 隻、是「**從個人英雄轉組隊配合**」的轉折段。

---

## 第三階：頂級威脅（終焉之章）

「**面對真正的 apex predator**」階段——但 apex 不是某種生物、是**人類文明**：

| 威脅 | 形式 | 給玩家的感受 |
|---|---|---|
| 推土機 | 不會死、無法擊敗、只能逃 | 無力感 |
| 化學排水管 | 持續產生毒區、毀棲地 | 失去家園感 |
| 塑膠袋 | 會困住、不會殺、但讓你動彈不得 | 慢性毒害 |
| 玻璃瓶碎片 | 地形殺、踩到就傷 | 環境本身變兇器 |
| 流浪犬群 | 人類棄養、最兇但最可憐 | 倫理複雜感 |
| 滅火器 / 化學噴霧 | 大範圍 AOE、見即逃 | 末日感 |
| 人類本身 | 不會直接攻擊、但他們**經過時你必須躲**、被看到就被驅趕 | **物種地位感**——你在牠們眼中只是「害蟲」|

→ 這階段沒有「**打贏 boss**」、只有「**生存** vs **被驅逐**」。
→ 玩家會開始問：「**這遊戲我是要打誰？**」——答案就是設計意圖。

---

## 最終 boss = 文明本身

整個遊戲最後沒有「打敗 final boss」的場景。
取而代之：

```
玩家通關所有第一第二階 boss
玩家擊退所有可擊敗的威脅
玩家面對推土機 + 化學排水管 + 塑膠袋海
玩家用盡所有撿來的武器、贏了無數場戰鬥

最終畫面：
  鏡頭拉遠
  整個森林被剷平成建地
  你的 5 個主角站在工地邊緣
  夕陽下
  字幕：「**你贏了每一場戰鬥。但你輸了戰爭。**」

  ↓

  Roll credits.

  ↓

  彩蛋（NG+）：
    工地隔壁、另一片森林、另一群小動物正撿著新的人類垃圾
    遊戲循環開始
```

→ **這個結局比任何環保宣傳片有力 100 倍**。

|---|---|---|
| 第一階 | 13 | 50% |
| 第二階 | 8-10 | 30% |
| 第三階 | 6-8 | 18% |

→ 對 souls-like 規模算中等、約 30-50 小時通關。
