# 造物森林：主敘事

## 一句話 logline

> 森林食物鏈中下層的小動物、用人類遺棄的小垃圾打造裝備、抵抗各種巨大掠食者。
> **他們的力量、來自摧毀棲地的敵人。**

## 世界觀

- **時空背景**：當代森林、緊鄰人類聚落 / 工業區邊緣
- **生態**：完整食物鏈仍在運作、但人類垃圾大量入侵
- **主角群**：食物鏈中下層的 5 種代表生物
- **「造物」**：不是神話創世、是**底層生物把人類垃圾「再造」成武器**——這就是「造物森林」的雙關名

## 三層意象

### 表層：可愛動物冒險
- 5 隻萌物拿著手工武器打 boss
- 視覺上類似 Cult of the Lamb / Tunic / 動物森友會
- 玩家第一眼覺得「**好可愛**」

### 中層：類魂硬核戰鬥
- 死亡會掉資源（撿來的廢物會散落）
- 死亡懲罰真實、不會 hand-holding
- 通關靠技術 + 學習、不靠等級碾壓
- 玩家第二眼覺得「**這他媽好難**」

### 深層：環保反諷
- 主角們的武器 = 人類垃圾
- 主角越強 = 人類丟越多垃圾
- 最終 boss 不是某隻怪、是**人類文明本身**
- 通關後玩家可能會想「**我打敗了所有掠食者、但棲地還是消失了**」
- 玩家第三眼覺得「**我好像被罵了**」

## 反諷母題展開

```
你的劍是別人不要的美工刀片
你的盾是別人喝完丟的瓶蓋
你的披風是被風吹散的塑膠袋
你的法袍是疫情時棄置的口罩
你的弩是水果攤拋棄的塑膠叉子
你的吉他是主人逃難前最後做給你的紀念物
你的暗器是家庭主婦不要的針線

你越強、人類丟越多
你的世界越好戰、你越成功

但當你打敗了食物鏈頂端的所有掠食者
你發現森林正在被推土機剷平
你的武器源源不絕、你的家正在消失
```

## 跟玩家對話的設計意圖

不說教、不在 cutscene 講「環保很重要」。
而是讓玩家：
- 戰鬥中**親手撿垃圾組裝武器**——體感「**沒有人類就沒有我的武器**」
- 解鎖新武器時、看見來源（「這是 7-11 飲料杯蓋」、「這是過期口罩」）——產生**異樣感**
- 通關後棲地依然消失——產生**無力感**
- 無力感 = 玩家自己悟出 message、不是被灌輸

→ 這比任何環保宣傳片有效。

## 跟 LL「賣 vibes 不賣 tools」的對齊

LL 守財奴依依賣的是「鬆弛的安心感」。
造物森林賣的是「**用敵人的武器反抗、但贏不了大局**」的悲劇英雄美學。
兩者都不靠功能、靠情緒。

## 跟 souls-like 同類型的差異

| 遊戲 | 主題 |
|---|---|
| Dark Souls | 諸神黃昏、命運不可逆 |
| Bloodborne | 宇宙恐怖、人類無知 |
| Hollow Knight | 文明遺跡、孤獨 |
| Cult of the Lamb | 邪教反差萌 |
| Tunic | 解謎 + 探索 |
| **造物森林** | **環保反諷 + 弱者抵抗 + 萌物寓言** |

→ souls-like 領域的**主題空位**、目前無人占據。
