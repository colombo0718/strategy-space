# 小螳螂 擬態刺客

## 基本資料

| 項目 | 設定 |
|---|---|
| 物種綱 | 昆蟲（螳螂）|
| 職業 | 刺客 / 暗殺 |
| 食物鏈位置 | 中下、被蝙蝠 / 蜂虎 / 蜥蜴 / 鳥吃 |
| 特殊能力 | **擬態**（融入葉片 / 樹枝）|

## 裝備（人類垃圾再造）

| 部位 | 物件 | 原本是什麼 |
|---|---|---|
| 武器 | 縫衣針線 | 家庭主婦丟棄的針線盒 |

## 反差萌（這個角色設計最尖銳）

- **家庭主婦的工具** vs **刺客武器**——性別 + 階級反轉
- 「**縫補的針** → **刺殺的針**」雙重隱喻
- 螳螂本身就「拜佛狀 + 雌螳螂吃配偶」——**家庭主婦黑暗面**的天然 metaphor

## 玩法 mechanic（草稿）

- 刺客定位：高傷單體 + 隱身 + 一擊脫離
- **擬態**：靜止在草叢 / 樹枝完全隱形、但移動就破除
- **縫衣針**：高暴擊單體、可投擲
- **縫線**：可拉繩設陷阱、或拉敵人靠近
- **背刺**：擬態狀態下接近敵人、第一擊必殺小怪
- **連針合擊**：跟其他角色配合、用線將敵人拖到陷阱位

## 性別 / 階級 metaphor（可選用、可不用）

如果遊戲走深度路線：
- 螳螂可設定為**母**——配偶被吃的家庭主婦
- 武器來源 = 家庭遺棄物 = 角色背景互相呼應
- 「**家庭主婦的暗器**」可以是一條 narrative DLC

如果遊戲走輕鬆路線：
- 不強調性別、就是個帥氣刺客
- 縫衣針線單純當「**被丟掉的小物件變強武器**」處理

→ 兩條都成立、看 colombo 哪天動工時定調。

## 天敵 boss

- 蝙蝠（夜行 + 超音波偵測、擬態無效）
- 蜂虎（專吃飛行昆蟲、空中對戰）
- 蜥蜴（陸地靈活、體型壓制）
- 大型蜘蛛（陷阱對決、誰能用線贏誰）

## 角色弧線（暫定）

- 起點：在某戶人家窗台被發現、針線盒成了家、後來人類搬走 / 房子拆遷
- 中段：加入主角群、提供潛行 + 偵察 + 暗殺
- 終點：面對推土機剷平住所、發現「**人類連我棲身的針線盒都不要了**」
