# 松松 松鼠大法師

## 基本資料

| 項目 | 設定 |
|---|---|
| 物種綱 | 哺乳類（松鼠）|
| 職業 | 法師 |
| 食物鏈位置 | 中下、被貓頭鷹 / 蛇 / 黃鼠狼 / 家貓吃 |
| 特殊能力 | **收集果實**（松鼠天賦、子彈來源）|

## 裝備（人類垃圾再造）

| 部位 | 物件 | 原本是什麼 |
|---|---|---|
| 法杖 | 金色咖啡湯匙 | 咖啡店附的金屬攪拌湯匙 |
| 法袍 | 破口罩 | 疫情時期棄置的醫療口罩 |
| 信使 | 小瓢蟲 | 森林原生、被松松馴服 |
| 彈藥 | 各種果實（松果 / 莓果 / 堅果）| 松鼠天然儲糧 |

## 反差萌

- **疫情遺物**（破口罩）vs **法袍**——口罩當披風、有種「**末日法師**」氣質
- **湯匙拋果實**= 把魔法降階成「**用湯匙當拋射器**」、可愛又物理
- 小瓢蟲信使 = 整個遊戲最萌的 mechanic

## 玩法 mechanic（草稿）

- 法師定位：遠程 + AOE
- **湯匙拋果實**：基本攻擊、依果實類型有不同效果
  - 松果 = 物理彈
  - 莓果 = 爆汁、潑灑範圍傷害 / 致盲
  - 堅果 = 高傷單體
  - 腐果 = 毒區
- **果實必須採集**——戰前要去松鼠藏匿點補貨
- **小瓢蟲信使**：可派出去偵察、回報敵人位置
- 破口罩**會破**——但材質特殊賦予**抗毒氣（化學污染）**屬性

## 天敵 boss

- 貓頭鷹（夜間活動 + 靜音飛行、教學被動偵測）
- 家貓（速度快 + 玩弄獵物、教學連續閃避）
- 黃鼠狼（鑽洞 + 體型小、無處可躲）
- 蛇（爬樹、松鼠的「家」被入侵）

## 角色弧線（暫定）

- 起點：森林一隅、發現咖啡店遺落的金湯匙、開始實驗
- 中段：認識主角群、提供遠程火力 + 偵察
- 終點：面對化學污染、發現自己的果實儲糧都不能吃（污染）、被迫覓食新區域
