# 金絲雀 吟遊輔助

## 基本資料

| 項目 | 設定 |
|---|---|
| 物種綱 | 鳥類（金絲雀）|
| 職業 | 吟遊輔助（Bard / Support）|
| 食物鏈位置 | 中下、被猛禽 / 流浪貓吃 |
| 背景 | **逃出鳥籠的家養金絲雀**、帶著主人最後的禮物 |

## 裝備（人類垃圾再造）

| 部位 | 物件 | 原本是什麼 |
|---|---|---|
| 武器 / 樂器 | 火柴盒小吉他 | **主人逃難 / 離世前、親手用火柴盒做給牠的禮物** |

## 情感深度（最揪心的角色）

```
火柴盒吉他 = 整個遊戲最有重量的物件
每次彈奏 = 主人還活著的記憶在發聲
吉他會磨損 = 主人的記憶在消逝
最終吉他可能會壞 = 玩家必須面對「**主人真的不在了**」的時刻
```

→ 如果遊戲設計成「金絲雀演奏可治療隊友、但每次都磨損吉他、最終會壞」、就會成為玩家**真的會掙扎要不要用**的機制。
→ 類魂遊戲就是要這種「**有重量的選擇**」。

## 反差萌

- **籠中嬌鳥** vs **荒野倖存者**
- 嬌弱外型 vs 戰場吟遊者
- 火柴盒吉他**小到可愛**、但戰場唯一治療來源

## 玩法 mechanic（草稿）

- 輔助定位：增益 / 治療 / debuff
- **演奏不同曲調**對應不同 buff（攻擊 / 防禦 / 速度 / 回血）
- **吉他耐久度有限**——玩家要選擇「**省著用 vs 救急**」
- **吉他壞掉後**：金絲雀失去戰鬥能力、變成普通鳥
  - 設計選項 A：直接 game over（永久死亡）
  - 設計選項 B：轉化為純偵察角色（飛行視角、無攻擊）
  - 設計選項 C：找新材料重做吉他、但音色不同、可能再也回不來
- **飛行**：可短暫飛行、為隊伍偵察

## 天敵 boss

- 紅尾鵟（猛禽、空中對戰）
- 流浪貓（家養出身、最熟悉的天敵）
- 蛇（蛋未孵化前的天敵、PTSD 元素）
- 籠中鳥同類（被人類養的同種、墮落版的「她可能變成的樣子」）

## 角色弧線（暫定）

- 起點：主人最後一刻打開籠子、給她火柴盒吉他、要她走
- 中段：加入主角群、用音樂凝聚團隊
- 終點：吉他壞掉的那一刻、玩家必須選擇——讓她退場、或重做新吉他（但不再是原本的記憶）

→ **這個角色獨自就可以撐起一個 DLC 故事**。
