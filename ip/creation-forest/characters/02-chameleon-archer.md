# 阿龍 變色龍盾弩手

## 基本資料

| 項目 | 設定 |
|---|---|
| 物種綱 | 爬蟲類（變色龍）|
| 職業 | 射手 + 重盾 |
| 食物鏈位置 | 中下、被鷹隼 / 狐狸 / 巨蟒 / 大型蜥蜴吃 |
| 特殊能力 | **隱身 / 擬態**（變色龍天賦）|

## 裝備（人類垃圾再造）

| 部位 | 物件 | 原本是什麼 |
|---|---|---|
| 武器 | 水果叉弩 | 餐桌用塑膠水果叉 |
| 防具 | 汽水罐大盾 | 鋁罐切開 + 拼接 |
| 箭袋 | 珍珠奶茶吸管 | 粗款珍奶吸管、裡面裝牙籤 |

## 反差萌

- **餐桌遺物**（水果叉 / 珍奶吸管）vs **戰場武器**
- 弱小爬蟲 vs 重盾射手
- **變色龍既能隱身又拿大盾**——主動隱身 + 被動防禦雙重生存策略

## 玩法 mechanic（草稿）

- 射手定位：遠程 + 防禦
- 牙籤箭可大量攜帶、但傷害低
- **隱身**：靜止不動可變色融入背景、躲掠食者
- 汽水罐盾：可格擋重擊、但材質薄、會凹陷
- **隱身射擊**：靜止狀態射擊有暴擊加成

## 天敵 boss

- 老鷹（從天而降、隱身對牠無效）
- 狐狸（嗅覺強、隱身對牠半失效、要逆風）
- 巨蟒（熱感應、隱身對牠完全無效——逼玩家學物理戰鬥）
- 紅尾鵟（飛行 + 視覺極佳）

→ 每個 boss 都教玩家「**隱身有極限**」、避免技能太萬能。

## 角色弧線（暫定）

- 起點：被人類當寵物棄養、學會用人類餐桌物件當武器
- 中段：找到其他 4 角色、提供隱身偵察支援
- 終點：發現「**人類連我的偽裝色 DNA 都改變了**」（化學污染導致變色能力退化）
