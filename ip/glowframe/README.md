## 一、定位

- **遊戲類型**：Warframe-like 共鬥 ARPG / 第三人稱動作（暫定）
- **玩家視角**：操作「戰甲」、意識可在多副戰甲之間切換
- **核心 mechanic**：戰甲組裝 + 武俠招式 + 光條視覺辨識
- **敘事母題**：**後人類 / 意識 vs 軀體 / 武俠永生**
- **對標作品**：Warframe（玩法）+ 創 光速戰記（美術）+ 樂園追放（世界觀）

## 二、跟造物森林的對比（LL 雙 IP 美學軸）

| 維度 | 造物森林 | **流螢戰甲** |
|---|---|---|
| 美學軸 | 接地、自然 | **離地、後人類** |
| 主角性質 | 食物鏈中下層生物 | **意識下凡的戰甲** |
| 武器來源 | 人類遺棄的垃圾 | **流光戰甲、武俠兵器** |
| 視覺風格 | voxel 萌物 + 真實物件 | **黑底 + 光條 + 武俠動作** |
| 敘事母題 | 環境反諷 / 弱者抵抗 | **意識永生 / 軀體即工具** |
| 玩法靈感 | Souls-like | **Warframe** |

→ **兩套互補不重複**、LL 旗下可同時跑、各打不同受眾。
