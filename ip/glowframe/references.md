# 流螢戰甲：對標作品深度分析

## 三大對標來源

| 作品 | 借鏡層面 | 不抄什麼 |
|---|---|---|
| **Warframe** | 玩法（共鬥 ARPG + 戰甲收集 + 任務制）| 不抄真免費 + 微交易模式 |
| **創 光速戰記**（Tron: Legacy）| 美術（黑底 + 光條）| 不抄賽車 / 圓盤戰 |
| **樂園追放** | 世界觀（後人類 + 意識上傳 + 戰甲下凡）| 不抄具體劇情 / 角色 |

---

## 一、Warframe（玩法）

### 借的東西

- **戰甲系統**：玩家擁有多副戰甲、切換玩
- **共鬥任務**：1-4 人 co-op
- **戰甲改造**：材料收集 + 升級 + 改裝
- **第三人稱動作**：跑酷 + 戰鬥流暢結合
- **任務循環**：星圖節點 / 不同類型任務

### 不借的東西

- ❌ **真免費 + 微交易模式**：LL 不擅長這種營運、且這條已紅海
- ❌ **科幻外星**：GlowFrame 走武俠 + 後人類、不是太空忍者
- ❌ **太空船 / 星圖**：地點是地球後人類社會、不需要星際導航
- ❌ **過度複雜的 mod 系統**：indie 規模做不來

### LL 怎麼差異化

- Warframe = 太空 + 忍者 + 科幻
- **GlowFrame = 後人類 + 武俠 + 哲學**
- 同樣是戰甲、但「為什麼穿戰甲」的世界觀完全不同
- 同樣是共鬥、但戰鬥節奏更慢 / 更注重武術美感

---

## 二、創 光速戰記（美術）

### 借的東西

- **黑底 + 光條**核心視覺語言
- **流光殘影**動作美學
- **極簡幾何**設計思維
- **電子化人類**的視覺隱喻

### 不借的東西

- ❌ **賽車 / 圓盤戰**：不適合武俠
- ❌ **電子世界內部**設定：GlowFrame 是現實世界、不是進電腦
- ❌ **80s 復古感**：要更現代化 / 武俠化

### LL 怎麼差異化

- 創 = 西方科技極簡 + 電子內部
- **GlowFrame = 東方武俠流光 + 現實後人類**
- 同樣黑底光條、但動作風格完全不同（武術 vs 賽車競技）

---

## 三、樂園追放（世界觀）

### 借的東西

- **意識上傳 + 數位永生**設定
- **戰甲下凡**機制（劇中主角 Angela 到地表也用戰甲）
- **後人類社會**結構
- **拋棄肉體的代價**哲學提問
- **殘體派**陣營（劇中地表派類似）

### 不借的東西

- ❌ **特定劇情 / 角色**：完全原創新劇情
- ❌ **特定戰甲設計**：GlowFrame 視覺完全自創
- ❌ **特定敵人 / 反派**：原創世界觀

### LL 怎麼差異化

- 樂園追放 = 動畫電影、單一線性劇情
- **GlowFrame = 遊戲、玩家自選角色 / 多分支**
- 樂園追放沒 emphasize 武俠、GlowFrame 用武俠當「**意識的審美表達**」

## 四、其他次要參考（風格 / 動作 / 哲學）

| 作品 | 借鏡點 |
|---|---|
| **Sekiro** | 武術節奏 + 完美閃避 + 武士魂機制 |
| **Genshin Impact** | 角色切換戰術 |
| **武林群俠傳 / 金庸群俠傳** | 武俠流派 + 角色養成 |
| **Cyberpunk 2077** | 後人類社會視覺氛圍 |
| **Ghost in the Shell** | 意識 vs 軀殼哲學深度 |
| **The Matrix** | 虛擬 vs 真實的母題 |
| **臥虎藏龍 / 一代宗師** | 武俠動作美學 |

## 五、市場定位差異化

```
科幻 ARPG 市場：
  Destiny 系列   = 太空 + FPS
  Warframe       = 太空 + 忍者 + 共鬥
  Anthem         = 已失敗
  The First Descendant = Warframe 後繼者、亞洲風
  ─────────────────────
  **GlowFrame** = 後人類 + 武俠 + 共鬥
                                ↑
                  這個交集目前無人占據
```

→ **GlowFrame 的差異化 = 武俠**——這條沒人做。

## 六、為什麼 LL 適合做這個 IP

1. **colombo 對武俠有興趣**（金庸古龍 / 武林群俠傳熟悉）
2. **colombo 美術品味偏極簡**（從 LL 視覺風格看出）
3. **voxel + 光條技術門檻低**（適合 indie / 一人開發）
4. **跟 LL agent 哲學自然對齊**（意識 / 軀殼 metaphor）
5. **跨域美學**（科幻 + 武俠）= 突破單一市場、可同時打東西方受眾

## 七、潛在合作對象（未來）

|---|---|
| **金庸基金會** | 武俠流派 IP 聯動（射雕戰甲 / 神鵰戰甲 等）|
| 古龍授權方 | 同上 |
| **武術指導 / 武打演員** | 動作捕捉 / 武術顧問 |
