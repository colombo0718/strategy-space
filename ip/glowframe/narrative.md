# 流螢戰甲：世界觀與敘事

## 一句話 logline

> 未來人類拋棄了肉體、將意識上傳至衛星伺服器。
> 需要與實體世界互動時、才會將意識下載至**通用戰甲**。
> **每個人都活在雲端、戰甲只是借來的軀殼。**

## 世界觀核心設定

### 後人類社會
- 肉體 = 過時的限制
- 意識 = 真實的「自己」
- 衛星伺服器 = 全人類的雲端居所
- 戰甲 = 跟物質世界互動的工具

### 戰甲機制
- 「**通用**」戰甲 = 任何人都可下載自己的意識進去
- 戰甲本身是工業量產品、不是個人化
- 個人化來自**意識操作風格 + 武術流派**
- 戰甲毀掉 = 換一台、意識不死
- 但有「**綁定戰甲**」（個人偏好調校）的傳統習慣

### 武俠永生悖論
- 武術原本是肉體技藝的累積
- 後人類社會肉體已棄、為什麼還練武？
- 答：**武術變成意識的審美 + 風格表達**
- 不再為了打贏、為了「**自己是誰**」
- 流派傳承靠意識師徒 mind-link、不是肉身教學

## 三層意象

### 表層：科幻 ARPG
- 玩家操控戰甲、共鬥任務、打怪升級
- Warframe-like 玩法
- 美術絢爛、動作華麗
- 玩家第一眼覺得「**好酷**」

### 中層：武俠流派
- 不同戰甲對應不同武術流派
- 招式名 / 配色 / 動作美學各有特色
- 玩家收集 / 切換不同戰甲體驗不同武術
- 玩家第二眼覺得「**這套打鬥很有東方韻味**」

### 深層：軀體哲學
- 「我」是意識還是身體？
- 軀體可換、意識永存——這是進化還是墮落？
- 戰甲毀掉時、意識會不會也有失落感？
- 「**綁定戰甲**」是執著、還是必要的自我認同？
- 玩家第三眼問自己：**「如果我能換身體、我會換嗎？」**

## 對位真實哲學

| 流派 | 對應 |
|---|---|
| 笛卡兒「我思故我在」 | 意識 = 真我 |
| 佛家「五蘊皆空 / 軀殼比喻」 | 戰甲只是借假修真的工具 |
| Ship of Theseus | 戰甲一直換、「**這還是我**」嗎？ |
| 後人類主義（transhumanism）| 整套世界觀的學術背景 |

## 主要 IP 衝突 / 故事鉤子

### 1. 戰甲記憶遺留
有些戰甲被用太久、會「**沾染**」前任使用者的習慣 / 偏好 / 甚至殘留意識片段。
你下次下載進去、會感覺到「**它記得別人**」。

### 2. 拒絕上傳的「殘體派」
仍有少數人類拒絕意識上傳、保留肉體。
被主流社會視為**野蠻 / 落後 / 危險**。
玩家可選擇接觸這群人、引發劇情衝突。

### 3. 衛星伺服器故障
某天衛星出問題、意識無法回到雲端、被困在戰甲裡。
**「永遠當戰甲」這個選項是 horror 還是 liberation？**

### 4. 通用戰甲反叛
戰甲是工業量產品、但 AI 已經夠強——
如果某天某個戰甲產生**自己的意識**、不想當「**借來的軀殼**」、怎辦？

→ 每一條都可以撐起一個 DLC 或續作。

## 玩家角色定位

預設玩家是「**新世代戰甲使用者**」：
- 剛從雲端下凡、第一次操作戰甲
- 邊玩邊學武俠流派
- 收集 / 改造 / 綁定喜愛的戰甲
- 在後人類社會探索「**我是誰**」

## 跟造物森林敘事的對比

| 造物森林 | 流螢戰甲 |
|---|---|
| 真實生物、真實死亡 | 虛擬意識、軀殼可換 |
| 弱者抵抗 | 強者選擇 |
| 環境污染壓迫 | 文明過剩抽離 |
| 主角會永久死亡（吉他壞掉的金絲雀）| 主角永遠不死（除非衛星壞） |
| 反諷「**我們對自然做了什麼**」 | 反諷「**我們對自己做了什麼**」 |

→ 兩 IP 主題不互斥、是 LL 對「**人類處境**」的兩個切面。

## 跟 LL agent 哲學的呼應

```
流螢戰甲世界觀：
  意識在衛星 + 需要時下凡至通用戰甲

LL agent runtime：
  agent 人格在 character bible / jsonl
  + 需要時 spawn 至 session / 軀殼跑

→ LL agent 不就是「流螢戰甲」的工程實作？
→ 流螢戰甲不就是 LL agent 的詩意版本？
```

**意外的對齊**：colombo 5 年前的科幻 IP、跟現在 LL agent 架構**用同一個 metaphor**。
這條對位**可能是 LL 對外講 agent 哲學的最美 talking point**。
