# 流螢戰甲：玩法草稿

> 大方向參考 Warframe、融入武俠 + 後人類元素。
> 細節未定、未來真要動工才展開。

---

## 一、玩法定位

| 項目 | 設定 |
|---|---|
| 遊戲類型 | 第三人稱共鬥 ARPG |
| 玩家數 | 單人 / 1-4 人 co-op |
| 視角 | 第三人稱（戰甲背後）|
| 節奏 | 中快、武俠動作 + 流光特效 |
| 主要靈感 | Warframe（任務 + 戰甲收集 + 動作）|
| 次要靈感 | Sekiro（武術節奏）+ Genshin（角色切換）|

## 二、核心 mechanic

### 1. 戰甲切換
- 玩家擁有「**意識**」、可下載至多副戰甲
- 出任務前選 1-3 副戰甲（看任務人數）
- 任務中可在戰甲間即時切換（戰術組合）
- 概念類似 Genshin 角色切換、但每副戰甲玩起來差更多（武俠流派不同）

### 2. 武俠招式系統
- 每副戰甲對應一個武俠流派
- 流派決定：基本動作 / 連招 / 必殺技 / 動作風格
- 例：劍俠流（範圍揮砍）/ 刺客流（高暴擊單體）/ 武僧流（拳腳 + 反擊）/ 弓箭流（遠程）

### 3. 流光特效作為戰鬥反饋
- 招式釋放 = 光條軌跡
- 暴擊 / 完美閃避 = 光條閃亮加強
- 受傷 = 光條黯淡 / 閃爍
- 玩家看光條 = 知道戰況（**美術即 UI**）

### 4. 戰甲改造 / 綁定
- 收集材料改造戰甲（光條顏色 / 武器配件 / 屬性）
- 長期使用某副戰甲 = 「**綁定**」加成
- 綁定戰甲毀掉時、需要重新調校新戰甲

### 5. 意識狀態
- HP = 戰甲耐久度（毀掉換新）
- 「**意識整合度**」= 真正的生命值
- 戰甲毀太多次、意識整合度下降、需要回衛星「**冥想**」恢復
- 整合度過低 = 暫時無法下載至戰甲、強制休息

## 三、任務結構（Warframe-like）

```
玩家在衛星意識社群接任務
  ↓
選戰甲（1-3 副、依任務難度）
  ↓
下載意識至戰甲、降臨任務地圖
  ↓
共鬥推進（co-op 1-4 人）
  ↓
完成任務、回收材料
  ↓
意識返回衛星、戰甲歸還或綁定
  ↓
改造 / 升級戰甲 / 收集新流派
```

### 任務類型
- **歷練**：簡單地圖、教學 / 練習
- **討伐**：消滅 boss
- **守護**：保護目標
- **滲透**：暗殺型、推刺客戰甲
- **遺跡探索**：撿前文明遺物 / 殘體派區域
- **意識救援**：救出陷在戰甲裡無法回衛星的同伴

## 四、世界觀帶來的玩法獨特性

### 1. 死亡 = 換戰甲、不是 game over
- 比 Souls-like 寬容（不損失進度）
- 但「**意識整合度**」是真懲罰、累積太多次失敗會被強制休息

### 2. 戰甲記憶 = 隱藏內容
- 某些戰甲被前任使用者用太久、會「**沾染**」前任意識片段
- 下載進去會看到別人的回憶、隱藏劇情
- 收集戰甲 = 收集前人故事

### 3. 殘體派 = 反派陣營 / 灰色道德
- 殘體派視主流為墮落、發動反抗
- 他們的戰士有真的肉體、會真的死
- 玩家面對他們會有道德掙扎（贏了會永遠殺死一個人）

### 4. 衛星伺服器危機 = 終局
- 衛星老化 / 被攻擊 / 故障
- 全人類意識面臨「**真死**」威脅
- 終局玩家要選擇：保衛衛星 vs 接受意識下凡 vs 跟殘體派合流

## 五、變現模式（粗估）

| 模式 | 說明 |
|---|---|
| 一次性購買 | 主遊戲 30-60 USD |
| 戰甲套裝 DLC | 新流派 / 新顏色系列、5-15 USD |
| 武俠特典 | 跟知名武俠 IP 聯動（金庸 / 古龍 / 武林群俠傳）|
| 美術圖鑑書 | colombo 寫書系列的延伸（vibe-to-orchestration 後續）|

→ **不走 Warframe 那種真免費 + 微交易**——LL 不擅長那種營運模式。
→ 走**一次性購買 + 體面 DLC** 路線、適合 indie 規模。

## 七、跟造物森林玩法對比

| | 造物森林 | 流螢戰甲 |
|---|---|---|
| 視角 | 第三人稱（小動物）| 第三人稱（戰甲）|
| 節奏 | 慢、souls-like | 中快、ARPG |
| 死亡感 | 沉重（永久 + 掉資源）| 輕（換戰甲、保意識）|
| co-op | 5 角色團隊 | 1-4 人 |
| 收集 | 垃圾 → 武器組裝 | 戰甲 → 流派切換 |
| 終局 | 失敗（環境贏）| 選擇（多結局）|

→ 兩款互補不重複、適合不同心情玩家。
