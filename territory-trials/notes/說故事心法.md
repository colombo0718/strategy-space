## 一、好故事的核心：代價之下的選擇

所有留得下來的故事，底層都是同一件事：**一個角色在乎某樣東西，而世界逼他為此付出代價**。

- 事件（沉船、戰爭、奪寶）只是**壓力容器**；重要的是壓力把角色逼到牆角時，他**選了什麼、放棄了什麼**。
- **選擇揭示人格，代價賦予重量。** 沒付過代價的勝利，觀眾不在乎。

### 硬判準一：兩難，不是困難
- 困難＝「怎麼打贏惡龍」→ 觀眾看熱鬧。
- 兩難＝「屠龍的劍要用你最愛的人來鑄」→ 觀眾睡不著。
- 《魔法公主》全片沒有一個乾淨的選項。**TT 北極星「沒有壞人，只有活下去」天生是兩難製造機**——遊戲哲學與敘事野心是同一個東西。

### 硬判準二：意料之外、情理之中
- 翻轉要讓觀眾驚呼，但回頭看每一步都埋好了。
- 羅生門中折即此招：村莊版本的每個「暴行」，在狼群版本裡都情理之中。

## 二、結構模板（存在、有名有姓）

| 模板 | 出處 | 一句話 |
|------|------|--------|
| 三幕劇 | 亞里斯多德 → Syd Field | 建立→對抗→解決，最通用骨架 |
| 英雄旅程 | Campbell → Vogler | 啟程→試煉→歸返，12 站，好萊塢標配 |
| **起承轉合** | 東亞古典 | 不靠衝突、靠「轉」的視角翻新——**羅生門是這一系** |
| Story Circle | Dan Harmon | 舒適區→渴望→陌生域→付代價→帶著改變回家，8 步 |
| Save the Cat | Blake Snyder | 15 節拍精確到頁碼，商業片配方 |
| 故事形狀 | Kurt Vonnegut | 主角命運曲線（Man in Hole／Cinderella…），情緒高低圖 |

TT pilot《狼王與村長》用的是**起承轉合＋羅生門雙版本**，不是英雄旅程——衝突不是要被「解決」的，是要被「看懂」的。

## 三、角色成長軌跡：一條收斂得很狠的公式

幾乎所有名著主角都能塞進這一條：

```
謊言 → 慾望 → 懲罰 → 剝奪 → 覺醒 → 抉擇
```

1. **謊言**：主角開場相信一個關於世界或自己的謊（白牙：「世界只有牙與爪」）。
2. **慾望（Want）**：他追逐一個表層目標；但真正需要的（**Need**）是另一樣。Want vs Need 的落差＝角色引擎。
3. **懲罰**：舊策略被世界反覆打臉。
4. **剝奪**：被剝到一無所有（all is lost 時刻）。
5. **覺醒**：這才看清謊言。
6. **抉擇**：高潮處，用新的自己做一次舊的自己做不到的選擇。

### 三種弧線變體
- **正變弧**：如上，主角蛻變（多數成長小說）。
- **悲劇弧**：同一條線的失敗版——到最後仍抱著謊言不放（蓋茨比）。
- **平弧**：主角不變、他是量尺，**改變的是身邊的世界**（福爾摩斯、導師型角色）。

### 套到 TT
- **狼王＝平弧**：他不需要成長，他的存在逼所有人成長。
- **村長＝成長弧候選**：從「妖獸＝災害」到看見對岸的守護者。
- **西頓式獵人**（未來資料片）：除妖英雄→保育者，最強的正變弧原型——真實世界的西頓因狼王洛波之死轉身成為保育運動先驅，「獵人被自己的獵物改變」。
- 羅生門群像劇本身偏「立場劇」，**弧線工具用在單角色深挖的卷**。

## 四、模板的真相：驗屍報告，不是食譜

> **模板為什麼準？因為它們是從已成功的故事屍檢歸納出來的。它能告訴你「好故事長這樣」，不能替你生出好故事。**

- 套公式寫作的下場＝「結構滿分、無人在乎」。
- 讀者記住的從來不是節拍，是**具體的細節與真實的痛感**：洛波循著白姐的氣味走進陷阱——這一幕不在任何模板裡，但它讓一個獵人變成保育之父。
- **正確用法**：寫的時候靠直覺與材料；卡住的時候拿模板當**診斷儀**——「觀眾為什麼不在乎？哦，主角還沒付過代價。」
- 一句話：**公式是醫生，不是母親。**

**資料策略結論**（同日討論定調）：敘事工藝存量足夠、不開「先蒐集」前置階段；真正要補的是——
1. **TT lore bible**（自產）：地名／譜系／規矩／境間地理，防資料片打架。
2. **真實生態案例**（每卷定向研究，可交 Gemini web search）：黃石狼群重引入、蒙古牧區人狼衝突、台灣石虎與雞舍——名著給骨架，**田野給體溫**，也保教育版科學準確。
3. **觀眾反應**（出片後才存在）：先射擊、再校準，不囤彈藥。
