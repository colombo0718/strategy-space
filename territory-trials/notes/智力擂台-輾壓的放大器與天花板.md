# TT 作為智力擂台：輾壓的四放大器與天花板

> **主題**：TT 這種複雜度的**對稱對局**，純靠「統御」的智力差（**無數值、無地形不對等**），能否形成「智力輾壓」？
> **沉澱自**：2026-07-20 colombo × 月衍卿 討論。為 SS agent 對戰 / RR benchmark 備料。
> **關聯**：[設計綱要](../設計綱要.md)「技術傳承」節、有靈三階（神魂層統御）、龍骨 MD→SP→TT→WW。

---

## 一、命題

給兩個**不同強度的 agent** 各自「統御」一方——例如兩個 agent 分別當兩座**對稱村落**的村長，或兩支**對稱狼群**的首領。設定完全對稱：沒有任何數值、地形、資源的先天不對等。**單憑決策品質的差距，能不能打出一面倒的輾壓？**

## 二、判斷

**能，而且 TT 體質很適合。** 但要點破：輾壓**不是「複雜」本身送的**，是 TT 恰好具備的四個性質給的——外加一個設計前提（見第四節）。複雜是必要、非充分。

## 三、四個放大器（純智力為何能輾壓）

1. **確定性**——MUST 的 `resolve()` 是純函式，無骰、無 RNG。這是地基：像西洋棋、不像撲克，弱者**沒有運氣可躲**，決策差距不會被雜訊洗掉。
2. **滾雪球經濟**——肉食殺→變強→再殺；人類錢→兵→更多錢；領土/資源控制。**開局一點智力優勢被複利放大成失控領先**。這正是「輾壓」（不只是贏、是碾）的引擎。
3. **同時承諾**（詠唱👄→施放👉、不可取消）——你得在看見對方動作**之前**下注，於是變成**預判／讀心**。會建模對手（theory of mind）的強 agent，靠佯攻、誘敵、伏擊贏下每一次預測。最吃深層認知的一層。
4. **不完全資訊**（FOV／晝伏夜出／霧）——藏訊息就獎勵推理。強的一方「看得少、推得多」，專打對方盲區。

＋ **多單位協同**：統御一整群，把技術表現面再乘上去——鉗形、餌、聲東擊西，遠比操一個單位深。

## 四、天花板（兩個誠實的但書）

- **退化解會殺掉深度**：若存在一條又強又簡單的「必勝套路」（如固定的每夜梭哈突襲），兩個 agent 一旦都找到它就收斂，智力便不再有差。**輾壓的生死線＝策略空間不能被輕易解掉**（每套打法都要有代價與剋制）。這是設計責任，不是複雜度自動送的。
- **鏡像頂端會變平**：對稱＋確定性下，**兩個都接近最優**的 agent 傾向逼近平手／微差。所以輾壓是「**差距**」的現象——差距大→碾壓、差距小→刀鋒／和局。當差分器很好，但頂到天花板會飽和（對 benchmark 反而正常：能一路把 agent 排到頂）。

## 五、兩個對局的體質差異

| 對局 | 性質 | 輾壓程度 | 變異 |
|------|------|---------|------|
| **兩村長**（經濟鏡像）| 純優化、複利 | **最易輾壓** | 低（像 AoE build order，會經營者 out-produce/out-tempo，專罰對方工人曝露的低效）|
| **兩狼首領**（生存鏡像）| 存亡邊際管理 | 也輾壓 | 高（飢餓是刀鋒，一次誤判的失敗獵殺可觸發餓死的向下螺旋）|

## 六、往深一層：TT ＝ 乾淨的智力差分器

這題真正在問的是——TT 能不能把勝負**歸因於決策本身**。能。而這正好餵到 LL 命脈：**RR 練 agent、SS 公開驗證 AI**。

- **統御層測的是「戰略智力」**（規劃、讀心、推理）——那是 **LLM／神魂層**最拉得開差距的地方；跟 SP/RL 的反射本能是**兩種試場**。
- **設計含義**：TT 可做成 SS 的「AI 智力擂台」——對稱、確定、藏訊息、複利，四者合起來是量測「將才」的好料。
- **接龍骨**：這也預告了 WW（第四階·multi-agent 治理）——當統御者本身是 agent、擂台就從「測一個將」升成「測一整套治理」。

## 七、設計待辦 / 開放問題

- **保持策略空間非退化**（每套打法有代價與剋制）＝輾壓與差分力的生死線。
- 狼群鏡像：**共享獵物 vs 各自獵物**——共享會變成「干擾博弈」（搶食、封鎖、斷糧），更深也更兇。
- **對局長度要夠**，複利才有空間表現（三兩回合分不出將才）。
- 量測面：要能記錄「決策品質」代理指標（資源效率、預判命中率、單位存活率），供 SS 天梯/RR benchmark 讀取。
