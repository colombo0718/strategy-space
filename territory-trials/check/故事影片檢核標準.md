# 故事影片檢核標準 — 引流片 × 遊戲劇情片

> **用途**：TT 故事內容的驗收 checklist。每支影片產出後逐項打勾；不合格退回重寫。
> **沉澱自**：2026-07-21 colombo × story Claude（cockpit）。
> **前提定案**：引流故事影片與遊戲劇情影片＝**不同規則與架構、互有聯繫**（詳見 [docs/狼王起源-教學戰役與雙軌敘事策略](../docs/狼王起源-教學戰役與雙軌敘事策略.md)）。

## 〇、共通鐵律（兩種影片都必須全過）

- [ ] **北極星合規**：沒有絕對壞人——每一方立場成立、可自陳為守方；不妖魔化任何一方。
- [ ] **正史一致**：不與 lore bible / 既有 canon 打架（狼王箭疤年表、村莊史、境的地理）。
- [ ] **不要臉原則落畫面**：掠取單位不露臉；農夫有臉；🎩 在揭曉卷前永不露面。
- [ ] **生態誠實**：出現的動物行為生物學站得住（狼群戰術／虎紋偽裝／晨昏性）——教育版命根。
- [ ] **製作鐵律**：Qwen prompt 無字（文字後製疊）；角色一致性（pilot 固定 seed／系列 LoRA）；labs VRAM 守衛。
- [ ] **基調**：歐美奇幻/拓荒；干支與仙俠語不入 TT；詞彙表（beast/vermin、Gaia）。

## A. 引流故事影片（YT）— 按戲劇節拍驗收

- [ ] **30 秒鉤子**：開場不需背景知識即可看懂張力。
- [ ] **兩難不是困難**：核心衝突雙方都有理、都有代價，且代價**真的付了**（有角色真的失去東西）。
- [ ] **翻轉合格**：意料之外、情理之中——回頭看每一步都埋好了（羅生門中折標準）。
- [ ] **羅生門成立**：看完無法簡單指認壞人（留言吵起來＝滿分）。
- [ ] **伏筆錨點埋入不展開**：定案錨點出現（箭疤／「牠曾經也有兄弟」／見弓退半步／沿影子走），不依賴、不解釋。
- [ ] **獨立成立**：沒玩遊戲也是完整故事，不留「必須玩才懂」的洞。
- [ ] **導流出口**：結尾自然接「如果坐在那個位子上的是你——」＋遊戲連結；可剪 30 秒翻轉段當 short。
- [ ] **規格**：6–8 分鐘；約 14 幀油畫；單一說書人旁白；Ken Burns 輕動態（不走 AI 影片）。

## B. 遊戲劇情影片／關卡敘事 — 按機制節拍驗收

- [ ] **服務關卡推進**：每段劇情對應一個明確機制教學目標（一關、一招、一刀）。
- [ ] **情感理由**：新機制的引入有劇情動機（學隱蔽＝喪親），不是彈窗說明。
- [ ] **節拍讓位給玩**：劇情段短、可跳過、不擋操作；玩家是親歷者不是觀眾（你**就是**那隻幼狼）。
- [ ] **進度綁定**：劇情推進綁玩家實際學會的能力（學會圍獵才給圍獵後的戲）。
- [ ] **callback 雙門檻**：兌現引流片錨點時——沒看過 YT 也看得懂；看過的拿額外爽感。兩邊不可犧牲任一。
- [ ] **兩軌互鎖**：不與引流片衝突；至少留一個未來卷可回收的新鉤子（飛輪不斷檔）。
