# Strategy Space「玄機萬境」仙俠敘事映射總表  
Version 1.0 · LeafLune Universe

> 原則：  
> - 仙俠敘事僅限 Strategy Space（SS）使用  
> - XX / RR / LL / GG 全部維持科技語  
> - 唯一跨平台的仙俠元素：LL Coin ≡ 「靈玉」  
> - 敘事不影響技術理解，兩者 1:1 映射

---

## 🟦 1. 平台本體映射

| 技術名詞 | 仙俠敘事名 | 說明 |
|----------|-------------|------|
| Strategy Space | **玄機萬境** | SS 的正式中文名；試煉、對戰、競技的主舞台 |
| Game / Environment | **境 / 幻境 / 萬境之一** | 每款遊戲即一個「境」 |
| New Game / Environment Update | **虛空裂隙開啟 / 新境成型** | 新遊戲上線的敘事方式 |

---

## 🟩 2. 玩家端角色映射（僅限 SS）

| 技術名詞 | 仙俠敘事名 | 說明 |
|----------|-------------|------|
| 玩家 / User | **弟子 / 修士** | 進入玄機萬境挑戰的參與者 |
| 學校 / 教育機構 | **宗門** | 合作單位的敘事化呈現 |
| AI Agent（本地） | **靈寵** | 玩家私自訓練的 AI 模型 |
| AI Agent（SS 上傳後） | **靈將** | 通過審核、具備戰績的戰鬥 AI |

> **靈寵 → 靈將**  
> 表示模型完成上傳並正式具備出戰資格（本體不變，角色地位提升）

---

## 🟨 3. 系統功能映射（SS 內部敘事）

| 技術名詞 | 仙俠敘事名 | 說明 |
|----------|-------------|------|
| PvP | **修士對決** | 玩家 vs 玩家 |
| PvE | **挑戰將靈 / 破境試煉** | 玩家 vs AI |
| Gatekeeper AI | **鎮塔將靈** | 官方或高階 AI 守關者 |
| Ranking | **天榜 / 地榜** | 天榜＝AI 排名；地榜＝玩家排名 |
| Matchmaking | **幻境引路** | 進入戰局前的匹配流程 |
| Season | **試煉輪迴** | 每季循環的敘事化呈現 |

## 🟫 5. 貨幣系統（唯一跨平台仙俠詞）

| 技術名詞 | 仙俠敘事名 | 說明 |
|----------|-------------|------|
| LL Coin | **靈玉** | 全生態通用貨幣；唯一跨平台的仙俠元素 |

靈玉用途：  
- 對戰獎勵  
- 購買外觀 / 權限  
- 參賽費、活動獎勵  
- 企業合作獎勵  

---

## 🟪 6. 完整敘事流程（技術 → 仙俠映射）

1. 玩家在 XX / RR 訓練 AI  
   → *技術語：模型 / Agent*

2. 玩家取得可運作的 AI  
   → **仙俠語：靈寵**

3. 上傳至 SS 玄機萬境  
   → **靈寵獲「靈將」之名**

4. 進行 PvP / PvE  
   → **破境試煉、修士對決**

5. 挑戰 Gatekeeper  
   → **鎮塔將靈試煉**

6. 進入排行榜  
   → **登天榜 / 入地榜**

7. 參與企業合作活動  
   → **天界勢力開啟虛空境**

8. 表現優異  
   → **得天界垂青，原地飛升（真實世界的實習 / 錄取）**

---

## 🟦 7. 玄機萬境 標準敘事一句話（官方建議）

> **玄機萬境，萬境試鋒。弟子攜靈寵入境，破關得靈玉；  
> 天界勢力觀其行，悟道者可原地飛升。**

## 🟨 8. 原則性結論

- **仙俠語僅限 SS 使用（玄機萬境）**  
- **其他平台全面維持科技語（XX / RR / GG / LL）**  
- **靈玉為唯一跨生態敘事詞彙**  
- **所有仙俠與技術對應 1:1，互不衝突，完整落地**
