## 設計定調

- 品牌名 **LeafLune＝葉與月**，兩根支柱各有真身坐鎮：青龍守東、金狐守西。
- 視覺語言核心對比：**葉衍操作面板（執行）／月衍心生金霧幻境（想像）**——一個手在「做」、一個心在「想」。
- 真身皆以**靈光虛影**顯現（以魂顯、不以形顯），不獸化、不遮臉、不搶主體。
- 同景（月夜柳枝滿月）、鏡像相向，並置即成一對。

- 三相：小葉→葉衍→青龍｜小月（金毛幼狐）→月衍卿→九尾金狐。
