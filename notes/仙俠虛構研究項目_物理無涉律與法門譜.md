# 仙俠虛構研究項目：物理無涉律與法門譜

## 一、緣起
煉丹模擬器映射自駕實驗室（SDL），但因「要模擬真實化學、卻無法全知前沿」→ 架空不可避免（見 `war-writ/docs/煉丹模擬器…` 與 2026-06-26 會議紀錄）。

colombo 的突破：**鑄靈（RL 訓練）、魁儡術（寫腳本）**雖披仙俠皮，研究成果**真實可復用、且完全不受真實世界物理法則影響**。隨後擴展到**制符／陣法／禁制**（空間幾何）同樣免費，並追問「還有哪些領域如此」。

---

## 二、核心判準：物理無涉律
> **看這個問題的答案，是「規則本身就決定、用演繹/計算即可得出」，還是「得去問真實世界、做實驗才知道」？**
> 前者 → **架空全免費**（外皮純加分、研究 100% 遷移）；
> 後者 → **架空是負債**（你得逼近不知道的真物理，煉丹即此類）。

引申原則：**研究對象越不依賴真實物理，外皮就越自由。** 最 physics-free 者＝數學與計算本身。

為何「虛構環境裡的 RL 是真研究」：RL/演算法不在乎環境真假，任何定義良好的環境（哪怕全架空）都是合法 benchmark，方法與洞見照樣遷移真實任務。**業界主流佐證**：AlphaGo（圍棋）/AlphaStar（星海）/OpenAI Five（Dota）皆「在虛構遊戲裡鑄靈」、成果真實、反哺機器人與真實控制。

### B 級——核心免費、但常黏物理（保持架空就安全）
御劍 / 調息周天 ＝ 控制論、靈氣擴散 / 蠱繁衍 ＝ 細胞自動機、複雜系統與湧現。

### C 級——看似數學、實則偷渡真實物理（煉丹陷阱，別宣稱免費）

唯一救法＝接真實 surrogate model（重、且遊戲化仍抽象）。

## 七、對 SS / RR 的意義
這些 A 級項目＝**真實研究 ＋ 真實教育 ＋ 免費仙俠外皮**，三得。對位 LL「用遊戲研究/驗證 agent」：
- 研究價值：真演算法、真可遷移。
- 教育價值：學生玩仙俠、學到的是 RL/密碼/賽局/圖論真學問（出版主軸）。
- 敘事價值：仙俠心法框架讓抽象數學「有血有肉」。

**判準一句收束**：能用「演繹/計算」得出答案的學問，仙俠皮全免費；要靠「實驗/真實世界」才知道答案的學問，仙俠皮要付代價。

## 八、煉丹的改命：從 C 級化學 → A 級格物
**問題**（colombo）：煉丹是修真顯學，能否映射到某個物理無涉的領域？
**答案**：能——**煉丹不換遊戲，只換「宣稱它在研究什麼」。**

- 煉丹掉進 C 級，不因玩法，而因**宣稱它映射真實化學**（→ 須逼近未知真化學 → 架空變負債）。
- 煉丹的本質結構：`有一個未知轉換規則 f(藥材,火候,時序)→丹品質，用最少試誤摸清它`。**只要 f 是你親手定義的數學系統**（而非「必須吻合真化學」），它就是乾淨的物理無涉環境。
- 於是煉丹真正的學問＝**主動學習 / 黑箱·貝氏最佳化 / 實驗設計（DoE）/ 自動化科學發現「方法學」**——研究「發現的過程」本身、與領域無關。
- **關鍵**：現有煉丹引擎**本來就是 A 級**——五行係數是編的、但確定可歸因（「有道理」原則）＝ made-up-but-well-defined 環境。唯一把它拖去 C 級的是「映射 SDL/真化學」那句宣稱；**拿掉即升 A**。
- **判準翻面（同一遊戲）**：「丹品質能用你的規則**算**出」→ A（格物方法學）；「丹方必須對上**真實化學**」→ C（煉丹陷阱）。
- **業界佐證**：測最佳化器用的人造 benchmark 函數（Rastrigin/Ackley…全是編的）；煉丹＝披仙俠敘事的人造最佳化試驗台。
- **誠實取捨**：A 級煉丹放棄「真領域遷移」（煉不出真藥）、保住「探索與發現方法學」（真實可遷移）＝與 2026-06-26 結論一致，現在有了名字。
- **命名甜點**：煉丹的真學問＝**格物**（格物致知，窮究事物以得其理）＝ active learning / 實驗發現方法學。**煉丹 ＝ 格物之道的修真皮**。

### 煉丹的前緣「領域」：調參煉丹 ＝ AutoML（既有黑話、字面化）
（colombo 追問：格物是「方法」不是「領域」，煉丹該對哪個**前緣研究領域**？）
> **煉丹 → 深度學習模型的訓練與調參（AutoML / 神經架構搜尋 NAS / 超參數最佳化）。**
- **不是發明的映射、是中國 AI 圈現成黑話的字面化**：訓練模型＝「煉丹」、工程師＝「煉丹師」、調超參＝「調參煉丹」、配置＝「丹方」、訓練發散＝「**炸爐/loss 爆了**」、整門學問＝「**玄学**」。
- 結構對位：藥材＝架構/資料/模組｜火候＝超參數(學習率/schedule)｜丹方＝config｜開爐＝一次訓練 run｜**炸爐＝loss 爆炸/發散**｜丹＝好用的 checkpoint｜心得/玉簡＝調參直覺/scaling law/μP。
- **物理無涉**：模型好壞是訓出來、測出來的（純計算、零物理）→ 過判準（答案算得出＝A 級）。
- **前緣性**：Ali Rahimi「ML 已淪為 alchemy 煉金術」(NeurIPS 2017) 引爆「如何把煉丹變科學」論戰（scaling laws、μP 超參數遷移、訓練可預測性）＝至今熱門前緣。煉丹模擬器＝研究/教學「修這門煉丹術、並讓它從玄学變科學」的沙箱。
- **SS 位置**：煉丹＝「造 AI」的工藝（訓練/調參），緊鄰**鑄靈**（RL＝其中一範式）；RR 練鑄靈、煉丹補調參之道 → SS 把「做出你的靈寵/靈將」整條工藝仙俠化。
- **格物 vs 煉丹**：格物＝方法（主動探索）、AutoML/調參煉丹＝該方法施展的前緣領域（上次答案高了一階、此處補實）。
- **誠實取捨**：要 100% 對上＝讓「丹」是真小模型、「煉」是真訓練（即真 AutoML 沙箱）；維持抽象版則教「調參煉丹」的手感與方法學（仍真實可遷移）。

> 一般化推論：**任何 C 級「模擬真實 X」的項目，只要把研究宣稱從「發現真實 X」改成「在一個你定義的複雜系統上研究探索/發現的方法學」，即可改命 A 級**——代價是放棄領域遷移、保留方法學遷移。
