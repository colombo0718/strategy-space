## 這是什麼

**Strategy Space（SS）** 是 LeafLune 生態的競技大廳，仙俠敘事名為「玄機萬境」。

玩家（弟子）在 RR 訓練好的 AI Agent（靈寵）上傳至此，升格為靈將，進入各境（遊戲）對戰、守擂、解任務、登天榜地榜。SS 是整個 LL 生態的戰場層——AI 被公開驗證、玩家成長有出口、教育成果有舞台、研究資料在此回流。

仙俠敘事僅限 SS 使用，其他平台（RR/XX/LL/GG）維持科技語。唯一跨平台的仙俠詞：**靈玉**（= LL Coin）。

## 在 LL 生態中的位置

```
RR（訓練 Agent）→ SS（Agent 出戰、競技、驗證）
LL（身份/靈玉）→ SS（登入、榮譽、經濟）
XX（教學攻略）→ SS（學成後上場）
```

### 完整使用者流程

```
SS 首頁
  └── 遊覽遊戲列表
        └── 點某個遊戲 → 看該遊戲的賽事列表
              └── 點某個賽事 → 報名
                    ↓
              切換進入 RR（自動載入該遊戲）
                    ↓
              上傳策略檔（Q-table / .pp）
                    ↓
              比賽開始，結果回寫天梯
```

**SS 是前台（遊覽 / 報名 / 排名）**，**RR 是執行引擎（遊戲載入 + 策略執行）**，用戶從 SS 無縫切入 RR，感覺像同一個系統。

### 教室 vs 賽事——同一套底層

SS 的「賽事」和 RR 的「教室」本質是同一個 Group/Room 機制：

| | RR 教室 | SS 賽事 |
|--|---------|---------|
| 建立者 | 老師手動建 | SS 平台自動建 |
| 加入方式 | 學生輸入班級代碼 | 玩家報名後自動推入 |
| 脈絡 | 學習導向 | 競技導向 |
| 一個遊戲可開幾個 | 多個教室 | 多個賽事 |

SS 可為同一個遊戲配置多個賽事（不同賽季、不同規則、不同組別）。
